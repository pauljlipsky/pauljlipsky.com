Get Started

How to Install a Skill

It takes about 30 seconds. Here's the process.

1

Download the .skill file

Each skill below has two download options. Start with the .skill file — it's the recommended format and works with the built-in skill uploader.

2

Open Claude Cowork

Launch Claude on your desktop and start a Cowork session (or open an existing one).

3

Go to Customize → Skills

Click the Customize button, then navigate to the Skills tab.

4

Upload the skill

Click upload and select the .skill file you downloaded. That's it — the skill is now active and ready to use.

If the .skill file doesn't work

Some setups may have trouble with the .skill format. If that happens, download the .md (Markdown) version instead and upload that the same way. It contains the exact same instructions — just in a different file format.

The Skills

Each one is a complete, standalone instruction set. Download what you need.

Brain Dump

Dump a messy stream of thoughts, ideas, and half-baked plans — Claude catches everything and organizes it into projects, action items, decisions, and a parking lot. Nothing gets lost.

Email Setup

One-time setup that interviews you about how you handle email — your voice, priorities, patterns — then builds a complete knowledge base that powers the Email Triage skill.

Email Triage

Runs a full inbox scan — classifies emails, researches unknown senders, drafts replies in your voice, and delivers a clean report. Requires Email Setup to be run first.

Landing Page

Generates a premium, animated single-page HTML landing page from a text prompt. Full 3D CSS animations, GSAP scroll effects, mouse parallax — one prompt in, polished website out.

Last 30 Days

Deep research agent that investigates any topic across Reddit, Hacker News, the web, and X/Twitter over the past 30 days. Returns a synthesized briefing with real citations and engagement data.

NotebookLM

Controls Google NotebookLM via browser automation. Read notebooks, add sources, generate Studio outputs (infographics, slide decks, audio overviews, study guides, mind maps, and more).

Take a Step Back

Mid-conversation reflection that zooms out from the weeds to reassess the bigger picture. Catches blind spots, questions assumptions, and resets direction when you're deep in the details.
